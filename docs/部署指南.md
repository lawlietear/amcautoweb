# AMC 文书生成器 - 部署指南

> 本文档面向系统管理员，详细说明系统部署步骤。
>
> 更新时间：2026-03-29

---

## 一、环境要求

- **操作系统**：Windows 10/11
- **Python**：3.8 或更高版本
- **网络**：局域网内可互通

---

## 二、部署步骤

### 步骤 1：安装 Python

1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.8+ 版本
3. **安装时勾选** "Add Python to PATH"
4. 完成安装

### 步骤 2：复制项目文件

将 `autodocweb_v2` 文件夹复制到目标电脑，确保包含：
```
autodocweb_v2/
├── app.py                  # 主程序
├── config/                 # 配置文件
├── core/                   # 核心模块
├── generated/              # 生成文件目录
├── static/                 # 静态资源
├── templates/              # 页面模板
├── word_templates/         # Word模板（重要！）
└── requirements.txt        # 依赖清单
```

### 步骤 3：安装依赖

打开 PowerShell 或 CMD，进入项目目录：
```powershell
cd autodocweb_v2

# 安装依赖
pip install flask docxtpl pandas openpyxl waitress
```

### 步骤 4：配置防火墙（重要）

以管理员身份运行 PowerShell：
```powershell
# 添加 5000 端口入站规则
netsh advfirewall firewall add rule name="AMC文书系统" dir=in action=allow protocol=tcp localport=5000
```

### 步骤 5：启动服务

```powershell
python app.py
```

启动成功后会显示：
```
============================================================
AMC 文书生成器 - Web 服务
============================================================

访问地址：
  - 本机：http://127.0.0.1:5000
  - 局域网：http://<本机IP>:5000

按 CTRL+C 停止服务
============================================================

[INFO] 使用 Waitress 生产服务器启动，支持多人同时访问
```


## 四、常见问题

### Q1：提示 "pip 不是内部命令"
**解决**：Python 安装时未勾选 "Add to PATH"，重新安装并勾选。

### Q2：同事无法访问
**排查步骤**：
1. 确认防火墙已放行 5000 端口
2. 确认同事和服务器在同一网段（IP 前三段相同）
3. 尝试在同事电脑上 `ping 服务器IP`

### Q3：模板找不到
**检查**：`word_templates/` 文件夹是否存在，模板文件（.docx）是否在文件夹内。

### Q4：中文显示乱码
**解决**：Windows 区域设置 → 管理 → 更改系统区域设置 → 勾选 "Beta: Unicode UTF-8"

## 五、生产环境建议

### 使用 Waitress 启动（推荐）
```powershell
# 后台运行（Windows）
pythonw app.py
```

### 开机自启动
1. 创建 `start.bat` 文件：
```batch
@echo off
cd /d C:\path\to\autodocweb_v2
python app.py
```

2. 将 `start.bat` 放入 Windows 启动文件夹：
```
Win + R → shell:startup → 粘贴 start.bat
```

## 六、技术参数

| 配置项 | 值 | 说明 |
|-------|-----|------|
| 服务器 | Waitress | 支持多人并发 |
| 端口 | 5000 | 可修改 |
| 线程数 | 4 | 支持 4 个并发请求 |
| 模板路径 | 自动适配 | 相对路径转绝对路径 |
| 输出目录 | generated/ | 自动创建 |
