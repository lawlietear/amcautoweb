| **默认值管理** | 保存甲方信息、违约金比例等固定字段 |

## 3. 数据结构设计

### 3.1 业务部变量（30个）

**批复信息表（9列）**：
- 基准日、债权本金金额、债权利息金额、其他费用金额
- 债权总额、保证金金额、转让价格、剩余交易价款、资金占用费率

**合同信息表（21列）**：
- 乙方名称、法定代表人、注册地
- 合同编号、收款银行、银行账户名称、银行账号
- 甲方联系人/电话/传真/地址/邮编/邮箱
- 乙方联系人/电话/传真/地址/邮编/邮箱
- 资金占用起算日、违约金比例（10.3.2条款）、违约金比例（10.3.3条款）

### 3.2 风险合规部变量（24个）

- 批复编号、会议日期、会次、业务分类、项目名称
- 审核人员、尽调报告名称、部门、项目负责人
- 债务人名称、户数、基准日
- 债权本金金额、债权利息金额、其他费用金额、债权总额
- 拟转让价格、交易费用承担主体、保证金金额、剩余交易价款
- 尾款支付期限、资金占用费率

## 4. 界面设计

### 4.1 首页

- **背景**：柔和网格渐变动画（iOS风格）

### 4.2 数据录入页

**业务部双表格布局**：
```
┌────────────────────────────────────────┐
│ 📋 数据录入    当前行数: 1    [+] [-]  │
├────────────────────────────────────────┤
│ 📋 批复信息（企业微信智能台账复制粘贴）  │

│ ┌─────┬─────┬─────┬─────┐            │
│ │     │     │     │     │  ← 行数同步 │
│ └─────┴─────┴─────┴─────┘            │
└────────────────────────────────────────┘
```

**行数控制**：
- 默认：1行
- 操作：点击`[+]`增加，`[-]`删除

## 7. 使用流程

### 7.1 业务部合同生成

1. **复制批复信息**：从企业微信智能台账复制，粘贴到批复信息表
2. **填写合同信息**：第一列乙方名称，对应上方行
3. **调整行数**：点击`[+]`添加多行（如需批量生成）
4. **保存默认值**：点击"保存为默认值"保存甲方信息
5. **生成文书**：点击"生成文书"，系统自动批量生成

### 7.2 风险合规部批复生成

1. **选择模板**：一次性/分期（自动识别）
2. **填写信息**：单表格模式填写所有字段
3. **导入Excel**：支持批量导入（可选）
4. **生成批复**：点击生成
